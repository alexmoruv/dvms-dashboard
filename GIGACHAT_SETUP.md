# 🔐 Настройка Gigachat API для dvms-dashboard

## Почему Gigachat?

✅ **Доступен в России** — без VPN и блокировок  
✅ **Русский язык** — полная поддержка кириллицы  
✅ **Бесплатный уровень** — достаточно для тестирования  
✅ **Безопасность** — API ключ не видит пользовательские данные  

---

## Шаг 1️⃣: Регистрация в Sberbank

1. Перейдите на **https://lk.sber.ru/**
2. Нажмите **"Вход"** или **"Регистрация"** (если нет аккаунта)
3. Авторизуйтесь через:
   - Номер телефона + SMS
   - ИЛИ e-mail + пароль
   - ИЛИ через Яндекс.ID

---

## Шаг 2️⃣: Получить API ключ

1. После входа в **lk.sber.ru** перейдите в личный кабинет
2. Нажмите на раздел **"Компании"** или **"Разработчикам"** (может быть в боку меню)
3. Найдите **"API"** → **"Gigachat"**
4. Нажмите **"Создать ключ"** или **"Получить API ключ"**
5. Вам предоставят:
   - **API ключ** (выглядит как длинная строка)
   - Опционально: Client ID, Client Secret

---

## Шаг 3️⃣: Добавить ключ в проект

### На вашем компьютере:

1. Откройте папку проекта `dvms-dashboard`
2. Создайте файл `.env.local` в корне (рядом с `package.json`):

```bash
# macOS/Linux
touch .env.local

# Windows (PowerShell)
New-Item -Name ".env.local" -ItemType "file"
```

3. Откройте `.env.local` в текстовом редакторе и добавьте:

```env
VITE_GIGACHAT_API_KEY=ВАШ_API_КЛЮЧ_ЗДЕСЬ
VITE_TAVILY_API_KEY=dummy_или_реальный_ключ
```

**Пример:**
```env
VITE_GIGACHAT_API_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiIxMjM0NTY3ODkwIiwibmFtZSI6IkpvaG4gRG9lIiwiaWF0IjoxNTE2MjM5MDIyfQ...
VITE_TAVILY_API_KEY=dummy
```

4. **Сохраните** файл (Ctrl+S)

---

## Шаг 4️⃣: Запустить проект

```bash
npm install
npm run dev
```

Приложение откроется на **http://localhost:5173**

## ⚠️ Важные замечания

### 🔒 Безопасность
- **Не коммитьте** `.env.local` в Git (добавлен в `.gitignore`)
- **Не делитесь** API ключом с другими
- Если ключ скомпрометирован — удалите его в lk.sber.ru и создайте новый

### 💳 Тарифы Gigachat
- **Бесплатный уровень:** ограничено тестовые объёмы
- **Платные планы:** от 500 до 100,000 токенов в день
- Проверьте тариф на lk.sber.ru → Биллинг

### 🌐 Tavily API (опционально)
Если хотите использовать веб-поиск:
1. Перейдите на https://app.tavily.com
2. Зарегистрируйтесь (бесплатно, 1000 запросов/месяц)
3. Скопируйте API ключ
4. Добавьте в `.env.local`:
```env
VITE_TAVILY_API_KEY=tvly-YOUR_KEY
```

## 🆘 Решение проблем

### Ошибка: "Ключ не найден. Проверьте файл .env"
✅ Убедитесь, что файл `.env.local` создан в корне проекта  
✅ Проверьте, что переменная точно названа `VITE_GIGACHAT_API_KEY`  
✅ Перезагрузите страницу браузера после создания `.env.local`

### Ошибка 401 Unauthorized
✅ API ключ скопирован неправильно (могут быть пробелы в начале/конце)  
✅ Ключ истёк — получите новый в lk.sber.ru

### Ошибка 429 Too Many Requests
✅ Превышен лимит запросов (зависит от вашего тарифа)  
✅ Подождите несколько минут или обновите тариф

### Приложение медленно работает
✅ Gigachat может быть перегружен — попробуйте позже  
✅ Проверьте интернет-соединение  
✅ Попробуйте меньший объём данных для парсинга
